## 慢图浏览测试版更新历史

<a href="/#/changelog">切换到正式版更新历史</a>

#### 0.1.51-beta05(2025-12-27)

1. 修复一些加载gif动画出现失败
2. 优化在移动/复制时创建新文件夹后直接进入该文件夹
3. 修复搜索照片可能的一些卡顿
4. 优化点击相册后进入照片列表后的加载速度
5. 无缝浏览支持双击放大图片后支持拖动照片查看

#### 0.1.51-beta04(2025-12-20)

1. 关于中新增分享应用给朋友
2. 修复一些情况可能造成加载图片空白的问题
3. 修复部分设备视频播放滑动控制音量失败的问题
4. 优化打赏新增PayPal渠道
5. 优化应用的部分英文场景下面的翻译 

#### 0.1.51-beta03(2025-12-20)

1. 优化移动照片时使用列表布局的相册路径显示
2. 修复日间模式下移动照片时使用列表布局相册路径无法显示的问题
3. 修复部分设备在排除无法选择外置sdcard
4. 修复浏览记录列表不显示视频长度的问题

#### 0.1.51-beta02(2025-12-13)

1. 支持双击直接播放视频，单击视频时长文字播放视频
2. 优化手机在内存不足时再次回到慢图丢失浏览进度的问题
3. 优化一些出现照片显示空白的问题
4. 优化设置中的幻灯片动画为翻页动画，翻页动画选项将应用于大图浏览时翻页动画
5. 优化回收站的排序方式
6. 优化隐私空间的一些卸载提示
7. 优化幻灯片播放中的一些动画延迟时间
8. 优化幻灯片中的随机播放模式
9. 移除一些设置项中多余的图标显示


#### 0.1.50-beta05(2025-11-15)

1. 修复启动应用的时候出现的卡顿(从0.1.50-beta03引入)
2. 修复部分设备使用隐私空间出现的一些崩溃
3. 修复设置中最大亮度选择时间范围后的显示错误
4. 优化应用内存占用
5. 设置底部属性增支持显示地理位置
6. 优化夜间模式下列表布局下的相册名称和文件名的字体颜色显示
7. 优化排除相册列表的缩略图宽高显示
8. 优化10W+数量级加载照片的速度
9. 设置新增实况图片允许自动播放
10. 增加照片浏览时翻页和播放实况图的振动效果
11. 优化浏览全图的时候在部分设备上出现的卡顿



#### 0.1.50-beta03(2025-11-02)

1. 修复隐私空间在部分设备的下滑手势跳过PIN码的校验
2. 修复相册长按的手势多选不一致的问题 
3. 修复实况照片在部分三星、小米设备播放失败的问题
4. 修复部分设备的照片出现时间线1970年的问题
5. 修复部分设备出现的手势返回导致退出应用的问题
6. 优化移动、复制操作，支持复制和移动到指定文件夹，支持记住最近操作的相册置顶等
7. 幻灯片支持随机播放


#### 0.1.49-beta03(2025-10-08)

1. 修复幻灯片播放偶发崩溃的问题
2. 修复播放视频在部分设备上出现程序停止响应的问题
3. 优化回收站、隐私空间右上角菜单的排列顺序
4. 照片列表支持文件名称显示(点击相册进入照片列表->菜单右上角显示文件名)
5. 照片列表支持打开照片所在文件夹(右上角文件夹)
6. 相册支持路径排序，支持列表布局
7. 修复无缝浏览不支持实况照片的问题


#### 0.1.49-beta01(2025-09-24)

1. 修复发送图片到桌面后无法查看的问题
2. 增加对实况图片标识符、虚拟相册增加实况图聚合
3. 优化Gif的加载、减少内存占用
4. 修复一些播放实况图失败的情况
5. 修复播放幻灯片可能崩溃的情况
6. 修复设置中使用系统媒体库收藏可能导致崩溃的情况
7. 优化在播放幻灯片自动隐藏旋转按钮
8. 优化应用在英语环境下的国际化翻译

#### 0.1.46-beta12(2025-08-30)

1. 升级安卓SDK到36
2. 修复搜索文件时输入特殊字符导致的崩溃
3. 修复文件传输助手偶发的崩溃
4. 修复部分设备图片加载错误的情况 

#### 0.1.46-beta11(2025-08-24)

1. 重构应用设置(修复升级版本后可能丢失配置的问题)
2. 搜索框UI显示优化
3. 修复主题模式在跟随系统的情况状态栏不可见的问题
4. 修复列数为1的情况显示的图片很小的问题
5. 修复布局方式为列表的情况下图片加载尺寸异常的问题 

#### 0.1.46-beta10(2025-08-21)

1. 设置新增中添加关闭左右手势滑动切换导航
2. 无缝浏览长按支持开启滚动条显示、支持关闭分割线显示
3. 修复由于列数为0导致的崩溃
4. 修复重命名相册导致的崩溃
5. 修复查看隐私空间相册照片时由于文件不存在导致的崩溃
